## 4. 문제점 및 해결 과정 (Issues & Solutions)
- **문제점 1**: 라이브러리 설치 명령어 오타 발생 (`ultralylistic`)
  - **현상**: `pip install ultralylistic` 입력 시 패키지를 찾을 수 없는 에러 발생 가능성.
  - **원인 및 해결**: `ultralytics` (YOLO 등 비전 AI용) 또는 XML 관련 라이브러리의 오타로 판단됨. `pip install ultralytics`로 명령어 수정 및 재설치 권장.
- **문제점 2**: 시스템 레벨에서의 pip 사용 (`sudo pip3`)
  - **현상**: `sudo pip3 install transforms3d` 실행.
  - **원인 및 해결**: Ubuntu 시스템 파이썬 환경을 직접 수정하는 방식은 패키지 충돌 위험이 있음. 향후 Python `venv` 가상환경 사용 또는 `--user` 옵션 사용 고려 필요.
